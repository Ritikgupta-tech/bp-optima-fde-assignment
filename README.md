# BP Optima FDE Assignment

## Problem

BP Optima serves multiple audiences:
- Chief Risk Officer
- CTO
- Investor
- Engineer

The current website communicates effectively but presents a largely uniform experience.

## Proposed Solution

An adaptive homepage that changes based on stakeholder intent.
